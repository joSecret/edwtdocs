Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

:::note
file with classe is in
`[main-doc-folder]/api/border.pcss`
in final this file is imported in
`[main-doc-folder]/api.pcss`
:::

## Additive

Add borders to custom elements:

<div class="not-content">
  <Tabs>
    <TabItem label="Example">
      <div class="d-flex gap-4">
        <div style="width: 100px; height: 100px" class="border"></div>
        <div style="width: 100px; height: 100px" class="border-top"></div>
        <div style="width: 100px; height: 100px" class="border-end"></div>
        <div style="width: 100px; height: 100px" class="border-bottom"></div>
        <div style="width: 100px; height: 100px" class="border-start"></div>
      </div>
    </TabItem>

    <TabItem label="HTML">
      ```html
      <div class="border"></div>
      <div class="border-top"></div>
      <div class="border-end"></div>
      <div class="border-bottom"></div>
      <div class="border-start"></div>
      ```
    </TabItem>

  </Tabs>
</div>

## Sizes

Use the scaling classes for larger or smaller rounded corners. Sizes range from 0 to 5, and can be configured by modifying the utilities API.

<div class="not-content">
  <Tabs>
    <TabItem label="Example">
      <div class="d-flex flex-wrap gap-4">
        <div style="width: 75px; height: 75px" class="bg-dark rounded-0"></div>
        <div style="width: 75px; height: 75px" class="bg-dark rounded-1"></div>
        <div style="width: 75px; height: 75px" class="bg-dark rounded-2"></div>
        <div style="width: 75px; height: 75px" class="bg-dark rounded-3"></div>
        <div style="width: 75px; height: 75px" class="bg-dark rounded-4"></div>
      </div>
    </TabItem>

    <TabItem label="HTML">
      ```html
      <div class="rounded-0"></div>
      <div class="rounded-1"></div>
      <div class="rounded-2"></div>
      <div class="rounded-3"></div>
      <div class="rounded-4"></div>
      ```
    </TabItem>

  </Tabs>
</div>

:::note
The variables used for this component.

| Name      | Default value     | Info                      |
| --------- | ----------------- | ------------------------- |
| --bw      | 1px               | Border width              |
| --bs      | var(--c-gray-300) | Border size               |
| --bc      | solid             | Border color              |
| --br      | 0.375rem          | Border radius             |
| --br-sm   | 0.25rem           | Border radius small       |
| --br-lg   | 0.5rem            | Border radius large       |
| --br-xl   | 1rem              | Border radius extra large |
| --br-pill | 100rem            | Border radius pill        |

:::
