Badges scale to match the size of the immediate parent element by using relative font sizing and em units. Badges don't have focus or hover styles for links.

## Use

:::note
To use this component in the TWIG file it must be imported.

```twig
{{ attach_library('YOURTHEME/badge') }}
```

:::

## Variables

:::note
The variables used for this component.
Can be used outside the tag "budget"

| Name       | Default value    | Info                          |
| ---------- | ---------------- | ----------------------------- |
| --badge-py | 0.35em           | Padding block (Top & Bottom)  |
| --badge-px | 0.65em           | Padding inline (Right & Left) |
| --badge-fs | 0.75em           | Font size                     |
| --badge-fw | var(--fw-medium) | Text color                    |
| --badge-c  | var(--c-white)   | Border color                  |
| --badge-br | var(--br)        | Border radius                 |

:::
