A generic close button for dismissing content like modals and alerts.

:::tip[Did you know?]
To use this component in the TWIG file it must be imported.

```shell
@import([YOUR-THEME]/close)
```

:::

import "../../../styles/edwt/components/close.pcss";

import { Tabs, TabItem } from "@astrojs/starlight/components";

## Example

Provide an option to dismiss or close a component with .btn-close. Default styling is limited, but highly customizable. Modify the Sass variables to replace the default background-image. Be sure to include text for screen readers, as we’ve done with aria-label.

<Tabs>
  <TabItem label="Design">
    <div class="bg-light p-4">
      <button type="button" class="btn-close" aria-label="Close"></button>
    </div>
  </TabItem>

  <TabItem label="HTML">
    ```html
    <button type="button" class="btn-close" aria-label="Close"></button>
    ```
  </TabItem>
</Tabs>

## Disabled state

Disabled close buttons change their opacity. We’ve also applied pointer-events: none and user-select: none to preventing hover and active states from triggering.

<Tabs>
  <TabItem label="Design">
    <div class="bg-light p-4">
      <button type="button" class="btn-close" disabled aria-label="Close"></button>
    </div>
  </TabItem>

  <TabItem label="HTML">
    ```html
    <button type="button" class="btn-close" disabled aria-label="Close"></button>
    ```
  </TabItem>
</Tabs>

## White variant

Change the default .btn-close to be white with the .btn-close-white class. This class uses the filter property to invert the background-image.

<div class="not-content">
  <Tabs>
    <TabItem label="Design">
      <div class="bg-primary p-4">
        <button type="button" class="btn-close btn-close-white me-2" aria-label="Close"></button>
        <button type="button" class="btn-close btn-close-white ms-2" disabled aria-label="Close"></button>
      </div>
    </TabItem>

    <TabItem label="HTML">
      ```html
      <button type="button" class="btn-close btn-close-white" aria-label="Close"></button>
      <button type="button" class="btn-close btn-close-white" disabled aria-label="Close"></button>
      ```
    </TabItem>

  </Tabs>
</div>

:::note
The variables used for this component.

| Name          | Default value | Info            |
| ------------- | ------------- | --------------- |
| --btn-c_s     | 14px          | Size            |
| --btn-c_p     | 6px           | Padding         |
| --btn-c_c     | var(--body-c) | Color           |
| --btn-c_bg    | var(--body-c) | Background      |
| --btn-c_bg--s | 14px          | Background Size |
| --btn-c_o     | 1             | Opasity         |
| --btn-c_br    | var(--br)     | Border radius   |
| --btn-c_img   | svg image     | SVG Image       |

:::
