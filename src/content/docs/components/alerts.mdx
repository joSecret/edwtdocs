Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

:::tip[Did you know?]
To use this component in the TWIG file it must be imported.

```shell
@import([YOUR-THEME]/alerts)
```

:::

:::note
The variables used for this component.

| Name                      | Default value             | Info                                                           |
| ------------------------- | ------------------------- | -------------------------------------------------------------- |
| --alert-py                | 1rem                      | Padding block (Top & Bottom)                                   |
| --alert-px                | 1rem                      | Padding inline (Right & Left)                                  |
| --alert-mb                | 1rem                      | Margin bottom                                                  |
| --alert-c                 | inherit                   | Text color                                                     |
| --alert-bg                | var(--bc)                 | Border color                                                   |
| --alert-bg-percentage     | 90%                       | white percentage                                               |
| --alert-br                | var(--br)                 | Border radius                                                  |
| --alert-b                 | 1px solid var(--alert-bc) | Border style                                                   |
| --alert-bc                | var(--alert-c)            | Border color                                                   |
| --alert-link-c            | color-mix                 |                                                                |
| --alert-link-c-percentage | 5%                        | If don't exist --alert-link-c, can be changed black percentage |
| --alert-heading-c         | inherit                   | Alert Heading color                                            |

:::
