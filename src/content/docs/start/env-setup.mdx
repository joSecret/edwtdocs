To get started with Eau de Web Theme, the only requirement is a Node Js environment.

How to install drupal theme.

## Installation theme

Composer command:
```command line
composer require 'drupal/edwt:^3.0'
```

- Read [Downloading and Installing a Theme from Drupal.org](https://www.drupal.org/docs/user_guide/en/extend-theme-install.html)
