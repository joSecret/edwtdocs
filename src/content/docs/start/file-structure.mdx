This section will show you how to keep your files organized. Our theme file structure that looks like this.

## Theme folder and file structure

```
├── example.json
├── index.html
├── index.js
├── package.json
├── package-lock.json
├── README.md
└── src
    ├── app.js
    ├── models.js
    ├── routes.js
    └── utils
        ├── another.js
        ├── constants.js
        └── index.js
```

```
+-- example.json
+-- index.html
+-- index.js
+-- package.json
+-- package-lock.json
+-- README.md
+-- src
|    +-- app.js
|    +-- models.js
|    +-- routes.js
|    +-- utils
|        +-- another.js
|        +-- constants.js
|        +-- index.js
+-- assets
|    +-- style.css
|    +-- cat.png
+-- example.com
```
